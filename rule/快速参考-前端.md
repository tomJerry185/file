# 前端规范快速参考卡 | Frontend Quick Reference

> 完整规范见《[前端规范](前端规范.md)》，本文档仅列出MUST级别核心规则。

## 组件规范

| 规则 | 要求 | 规范章节 |
|------|------|----------|
| API风格 | 必须使用`<script setup lang="ts">` Composition API | 3.1 |
| 组件命名 | PascalCase，文件名与组件名一致 | 3.1 |
| Props定义 | 使用`defineProps<T>()`类型声明 | 3.1 |
| 事件定义 | 使用`defineEmits<T>()`类型声明 | 3.1 |
| 状态管理 | 本地用ref/reactive，跨组件用Pinia | 3.4 |

## TypeScript规则

| 规则 | 要求 |
|------|------|
| 禁止any | 使用`unknown`替代 |
| 严格模式 | tsconfig开启strict全家桶 |
| 类型断言 | 优先使用类型守卫，禁止双重断言 |
| 导入顺序 | Vue核心→第三方→内部→相对→type |

## 设计系统

| 类别 | 使用方式 |
|------|----------|
| 颜色 | CSS变量 `--color-primary-500`，禁止硬编码 |
| 间距 | 8点网格 `--spacing-4`（16px） |
| 字号 | 阶梯制 `--font-size-md`（14px） |
| 圆角 | `--radius-md`（4px） |
| 阴影 | `--shadow-md` |

## 可访问性（MUST）

| 规则 | 要求 |
|------|------|
| 语义化HTML | 使用header/nav/main/section/footer |
| 图标按钮 | 必须有aria-label |
| 表单控件 | 必须有label或aria-label |
| 对比度 | 正常文本>=4.5:1 |
| 减少动画 | 支持`prefers-reduced-motion` |

## 性能指标

| 指标 | 目标值 |
|------|--------|
| LCP | <= 2.5s |
| INP | <= 200ms |
| CLS | <= 0.1 |
| FCP | <= 1.8s |
| 首屏JS | <= 250KB |

## 安全规则

| 规则 | 要求 |
|------|------|
| v-html | 必须用DOMPurify净化 |
| Token存储 | httpOnly cookie，禁止localStorage |
| 生产环境 | 禁止console.log输出 |
| 敏感数据 | 不在localStorage存储 |

## 测试要求

| 类型 | 覆盖率 | 框架 |
|------|--------|------|
| 单元测试 | 行>=80%，分支>=75% | Vitest + @vue/test-utils |
| E2E测试 | 关键路径 | Playwright |

## 工程化

| 工具 | 配置 |
|------|------|
| ESLint | vue3-recommended + typescript-recommended + a11y |
| Prettier | 单引号、2空格、100字符宽 |
| Stylelint | 标准配置 + 设计令牌检查 |
| 构建工具 | Vite，路由级代码分割 |
