# 全局规范速查卡 | Global Quick Reference

> 一页纸速查：所有领域最重要的 MUST 规则。新人快速上手 + 日常自查。
> 完整规范见各领域规范文件，快速参考卡见 `快速参考-后端.md` / `快速参考-前端.md` / `快速参考-中间件.md`。

---

## 后端 Top 5

| # | 规则 | 速记 |
|---|------|------|
| 1 | **构造器注入** | `@RequiredArgsConstructor` + final，禁止 `@Autowired` 字段注入 |
| 2 | **禁止 SELECT *** | Mapper XML 必须明确列出字段 |
| 3 | **Service 必须有接口** | `XxxService` 接口 + `XxxServiceImpl` 实现类 |
| 4 | **统一 Result\<T\>** | 所有接口返回 `{code, message, data, traceId}` |
| 5 | **错误码枚举** | `ErrorCodeEnum`，禁止魔法数字 |

## 前端 Top 5

| # | 规则 | 速记 |
|---|------|------|
| 1 | **script setup + TS** | `<script setup lang="ts">`，禁止 Options API |
| 2 | **禁止 any** | 用 `unknown` 或具体类型替代 |
| 3 | **设计令牌** | 颜色/间距/字号用 CSS 变量，禁止硬编码 |
| 4 | **v-html 必须净化** | 使用 `DOMPurify.sanitize()` |
| 5 | **清理 console.log** | 生产环境禁止 console 残留 |

## 中间件 Top 5

| # | 规则 | 速记 |
|---|------|------|
| 1 | **敏感配置环境变量** | 密码/密钥/连接串禁止硬编码 |
| 2 | **必须配超时** | 所有外部调用必须设连接+读取超时 |
| 3 | **缓存必须设 TTL** | 禁止无过期缓存，雪崩防护 +10% 随机偏移 |
| 4 | **MQ 必须幂等** | 消费端用业务唯一键去重 |
| 5 | **禁止嵌套** | 分布式事务与 MQ 消息不得互相嵌套 |

## 安全 Top 5

| # | 规则 | 速记 |
|---|------|------|
| 1 | **API 限流** | 登录5次/分、下单10次/分、通用1000次/分 |
| 2 | **禁止硬编码密钥** | 所有密钥从环境变量或密钥管理服务获取 |
| 3 | **v-html 用 DOMPurify** | 净化所有用户可控的 HTML 内容 |
| 4 | **CORS 禁止 *** | `allowed-origins` 必须白名单 |
| 5 | **Cookie HttpOnly** | Sa-Token 开启 HttpOnly + Secure + SameSite |

## 运维 Top 5

| # | 规则 | 速记 |
|---|------|------|
| 1 | **Runbook 必须有** | 每个服务必须有标准运维手册 |
| 2 | **容量基线** | 服务 QPS 基线 + 资源水位阈值 |
| 3 | **DB 变更审批** | DDL 必须走审批流程，禁止直连生产改表 |
| 4 | **灾备演练** | 每季度至少一次故障转移演练 |
| 5 | **告警闭环** | 告警必须有人认领 + 限时响应 + 事后 RCA |
